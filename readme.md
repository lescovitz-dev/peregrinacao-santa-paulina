# 11ª Peregrinação ao Santuário de Santa Paulina

Aplicação web em formato PWA para apoiar os peregrinos durante a 11ª Peregrinação ao Santuário de Santa Paulina.

## Funcionalidades
- Página de inscrição com PIX
- Checklist do peregrino (funciona offline)
- Cronograma da peregrinação
- Oração a Santa Paulina
- Mapa dos trajetos (Dia 1 e Dia 2)
- PWA instalável (Android e iOS)

## Tecnologias utilizadas
- Python 3
- Flask
- HTML, CSS e JavaScript
- Google Maps (embed)
- PWA (Service Worker + Manifest)

## Como rodar localmente
```bash
pip install flask
python app.py
